Caregent - AI Career Agent

0. 프로젝트 성격

Caregent는 완성형 서비스를 빠르게 만드는 프로젝트가 아니라, 데이터 엔지니어링 학습과 포트폴리오 증명을 위한 실습 프로젝트이다.

제품 비전은 AI Career Agent로 유지하지만, v1에서는 Airflow, ETL, PostgreSQL, 데이터 정제, 집계, 운영 로그 확인을 우선 학습한다.

Electron, 고도화된 AI Agent, RAG, 벡터DB는 v1 이후 확장 기능으로 둔다.

⸻

1. 프로젝트 개요

Caregent는 AI와 데이터 엔지니어링 기술을 활용하여 사용자의 취업 준비 과정을 지원하는 AI Career Agent 플랫폼이다.

단순히 채용공고를 수집하여 보여주는 서비스가 아니라, 실시간 채용시장 데이터를 분석하고 사용자의 현재 역량과 비교하여 맞춤형 취업 전략과 학습 로드맵을 제공하는 것을 목표로 한다.

사용자는 자신의 학력, 자격증, 기술스택, 관심 직무를 등록할 수 있으며, Caregent는 채용공고, 기술 트렌드, 기업 채용 동향을 지속적으로 수집·분석하여 사용자에게 필요한 역량과 학습 방향을 제안한다.

Caregent는 데이터 엔지니어링 기반 ETL 파이프라인과 AI Agent를 결합하여 사용자의 커리어 성장을 지원하는 지능형 취업 비서 서비스를 지향한다.

⸻

2. 핵심 목표

목표 1

채용시장의 최신 데이터를 자동으로 수집한다.

목표 2

사용자의 역량과 채용공고 요구사항을 비교한다.

목표 3

부족한 기술과 우선 학습 항목을 도출한다.

목표 4

AI Agent를 통해 개인 맞춤형 취업 전략을 생성한다.

목표 5

데이터 기반 커리어 의사결정을 지원한다.

3. 주요 기능

3.1 채용공고 수집

수집 대상

* 대한민국 대기업
* 대한민국 중견기업
* IT 계열
* 반도체 계열

수집 경로

* 기업 채용 페이지
* 채용 플랫폼
* 기술 채용 게시판

특징

* Airflow 기반 자동 수집
* 매일 오후 3시 자동 실행
* 중복 데이터 제거
* 신규 채용공고 자동 갱신

⸻

3.2 채용시장 분석

수집된 데이터를 기반으로 다음 정보를 분석한다.

* 기술스택 등장 빈도 분석
* 직무별 요구 역량 분석
* 기업별 채용 트렌드 분석
* IT / 반도체 분야별 시장 분석

예시

* 최근 3개월 AWS 수요 변화
* 데이터 엔지니어 직무 기술스택 분석
* 반도체 기업 채용 증가 추세

⸻

3.3 사용자 프로필 관리

사용자 등록 정보

* 학력
* 학점
* 자격증
* 기술스택
* 관심 직무
* 관심 산업군

사용자는 자신의 현재 역량을 지속적으로 관리할 수 있다.

⸻

3.4 역량 매칭

사용자의 프로필과 채용시장 데이터를 비교한다.

분석 항목

* 기술스택 매칭
* 직무 적합도
* 산업군 적합도
* 부족 역량 분석

결과 예시

* IT 분야 적합도 82%
* 반도체 분야 적합도 67%
* 부족 기술: AWS, Docker

⸻

3.5 Skill Gap Analysis

Caregent의 핵심 기능

사용자의 현재 역량과 채용시장의 요구 역량을 비교하여 부족한 기술과 학습 우선순위를 도출한다.

예시

현재 보유

* Python
* FastAPI
* PostgreSQL

시장 요구

* Python
* PostgreSQL
* AWS
* Docker
* Airflow

결과

부족 기술

1. AWS
2. Docker
3. Airflow

⸻

3.6 AI Career Agent

사용자는 자연어로 질문할 수 있다.

예시

“데이터 엔지니어가 되려면 무엇을 공부해야 할까?”

“현재 내 스펙에서 가장 부족한 역량은?”

“최근 채용시장에서 수요가 증가하는 기술은?”

“IT와 반도체 중 어느 분야가 나에게 더 적합할까?”

AI Agent는 수집된 데이터와 사용자 프로필을 기반으로 답변을 생성한다.
